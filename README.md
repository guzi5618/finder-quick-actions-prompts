# Finder Quick Actions Prompts

[中文说明](./README.zh-CN.md)

Small reusable prompts for building macOS Finder quick actions with AI.

This repo currently includes two tiny Finder helpers:

- `Show and Copy Full Path`
- `Open Terminal`

## What's inside

- [prompts/show-and-copy-full-path.md](/Users/hao/finder-quick-actions-prompts/prompts/show-and-copy-full-path.md)
- [prompts/open-terminal-here.md](/Users/hao/finder-quick-actions-prompts/prompts/open-terminal-here.md)
- [scripts/install_full_path_quick_action.sh](/Users/hao/finder-quick-actions-prompts/scripts/install_full_path_quick_action.sh)
- [scripts/install_open_terminal_quick_action.sh](/Users/hao/finder-quick-actions-prompts/scripts/install_open_terminal_quick_action.sh)

## Use cases

- Show a selected file or folder's full path and copy it to the clipboard
- Open Terminal in the selected folder, the selected file's parent folder, or the current Finder directory

## How to use

1. Copy a prompt from `prompts/`.
2. Give it to your AI coding assistant.
3. Generate and install the Finder quick action on your Mac.

The `scripts/` folder contains working reference implementations.
